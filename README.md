# Calculator

A modern, clean calculator with a light blue/grey gradient background. Built with **HTML**, **CSS**, and **Vanilla JS**.

## ✅ Features
- Responsive layout
- Beautiful typography (Inter)
- Smooth button interactions

## 🛠️ Tech
- HTML5
- CSS3 (Flex & Grid)
- JavaScript

## 🚀 How to Run
Open `index.html` in your browser.
